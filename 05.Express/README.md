## Middleware
미들웨어 예제

- middleware : 미들웨어 작성
- faviconAndLog : 파비콘과 로그 미들웨어
- responses : 응답 처리 함수
- methodOverride : HTML폼에서 Delete/Put 등의 메소드 사용하기

## Routing
라우팅 예제

- staticFile : 정적 파일 라우팅
- routingBasic : 라우팅 기본
- routingMethod : 라우팅 메소드
- routingParams : 라우팅 파라미터
- routingRegEX : 정규식을 이용한 라이팅
- app, eatingRouter, greetingRouter : 라우팅 모듈
